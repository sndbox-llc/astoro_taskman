<Card title="案件とタスクの紐づけ" icon="puzzle">

案件ごとにタスクを管理可能。これによりどの案件に対してどの程度のタスクが残っているのかを簡単に把握できます。もちろん、案件に紐づけない独立したタスクの作成も可能です。
案件とタスクの紐づけにより業務の効率化とミスの防止を実現します。

<Card title="カレンダーでタスクの締め切りを管理" icon="puzzle">

登録したタスクに期限を指定すれば、カレンダー上に自動でマッピング。「今日の予定」「今週やること」「今月のタスク」など、期限に基づいたタスクの把握が簡単になります。
チーム内の全員の予定を１つのカレンダーで全員が共有できるため、スケジュール調整もスムーズに行えます。

<Card title="弁護士向け機能" icon="puzzle">

弁護士事務所に特化した機能も充実。弁護士特有の裁判に関する記録を時系列に管理できます。案件と裁判を紐づけて、情報を一元管理。属人化を防止し、事務局と弁護士の連携を強化します。
[民事裁判書類電子提出システム Mints](https://www.mints.courts.go.jp/user/)の入力項目に対応しているため、Mintsへの入力もスムーズに行えます。
Mints以上に細かく、各裁判ごとの記録もしっかり残せます。
